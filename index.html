    Tags information
    head tag contains the title and the meta information of the page
    meta tag contains the meta information of the web page
    hr to add a horizontal line which act as a divider
    a to add a hyperlink, where the "href" will contain the path or link
    h(1,2,3,4,5,6) heading tag h1 will be the largest and h6 smallest
    img to add image in the page, where "src" contains the path or link
    table for creating table. Tables can also be used for basic formatting of elements
    thead specifies the header of the table
    tbody specifies the body of the table
    tfoot specifies the footer of the table
    tr specifies the row in table
    td specifies the data cell
    form for making form(to enter the data); important attributes are: actions which tells what will happen when form is submitted
    lable to add lable to form fields in the form
    input for inputting data(type can be defined in the type attribute,name can be used to classify the inputs
